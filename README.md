# Rugby Data

Extracting data on rugby games, creating visualizations, and building predictive models.

## Project Structure

- `src/scraping/` - Data extraction modules
- `src/visualization/` - Visualization utilities
- `src/modeling/` - Predictive modeling
- `notebooks/` - Jupyter notebooks for exploration
- `data/` - Local data storage (gitignored)
- `tests/` - Tests

## Setup

```bash
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```
